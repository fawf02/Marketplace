# StudentHub - Маркетплейс для студентов

Современное веб-приложение для покупки и продажи товаров среди студентов, разработанное на **Vue 3**, **Vite** и **Tailwind CSS**.

## 🚀 Технологический стек

- **Vue 3** - прогрессивный JavaScript фреймворк
- **Vite** - быстрый сборщик приложений
- **Vue Router 4** - маршрутизация
- **Tailwind CSS** - утилитарный CSS фреймворк
- **Composition API** - современный способ написания компонентов
- **`<script setup>`** - синтаксический сахар для Composition API

## 🎯 Описание основных компонентов

### Компоненты Навигации и UI

#### NavBar.vue
Навигационная панель с:
- **Логотип** (StudentHub)
- **Строка поиска** с кнопкой поиска
- **Навигационные ссылки**: Главная, Каталог
- **Иконка корзины** (связь с CartModal)
- **Иконка профиля** для входа
- **Sticky позиционирование** (остаётся вверху при прокрутке)
- **Адаптивное меню** для мобильных устройств
- **Активные ссылки** с подсветкой

### Модальные окна

#### LoginModal.vue
- Форма входа в систему
- Валидация электронной почты
- Интеграция с `authStore`
- Кнопка переключения на регистрацию

#### RegisterModal.vue
- Форма регистрации нового пользователя
- Проверка совпадения пароля
- Принятие условий использования
- Переключение на форму входа

#### ProfileModal.vue
- Просмотр и редактирование профиля
- Отображение информации пользователя
- Выход из системы

#### CartModal.vue
- Просмотр товаров в корзине
- Удаление товаров
- Расчёт итоговой стоимости
- Оформление заказа
- Связь с `cartStore`

#### SellModal.vue
- Форма создания нового объявления
- Загрузка фото товара
- Добавление описания товара
- Установка цены
- Выбор категории

### Страницы (Views)

#### HomeView.vue
Главная страница с:
- **Hero section** - приветственный блок с call-to-action
- **Features section** - основные преимущества платформы
- **Популярные товары** - сетка товаров
- **Промо секция для продавцов** (SellerPromo компонент)

#### CatalogView.vue
Каталог товаров с:
- **Фильтры слева**: категория, цена, рейтинг
- **Сетка товаров** справа
- **Сортировка** товаров
- **Поиск** товаров
- **Загрузка товаров по клику** (infinite scroll)

#### ProfileView.vue
Профиль пользователя с:
- **Информация о пользователе**
- **Список моих объявлений** (для продавцов)
- **История покупок** (для покупателей)
- **Отзывы и рейтинги**

#### LandingPage.vue
Лендинг страница перед авторизацией - первое впечатление от платформы

### Специальные компоненты

#### SellerPromo.vue
Промо компонент для привлечения продавцов:
- Описание преимуществ продажи
- Call-to-action кнопка "Начать продавать"

#### WelcomeItem.vue
Компонент для отображения преимуществ/фич в слайдере на главной

### Иконки (icons/)
Переиспользуемые SVG иконки компоненты:
- IconCommunity.vue
- IconDocumentation.vue
- IconEcosystem.vue
- IconSupport.vue
- IconTooling.vue

## 📦 Хранилище состояния (Stores)

### authStore.js
Управление состоянием аутентификации:
- Текущий пользователь
- Токен авторизации
- Статус загрузки
- Методы: login, register, logout, checkAuth

### cartStore.js
Управление корзиной товаров:
- Список товаров в корзине
- Количество товаров
- Итоговая стоимость
- Методы: addToCart, removeFromCart, clearCart, updateQuantity

## � Архитектура приложения

### Frontend (Vue 3 + Vite)
- **Фреймворк**: Vue 3 с Composition API и `<script setup>`
- **Сборщик**: Vite для быстрой разработки и оптимизированной сборки
- **Маршрутизация**: Vue Router 4
- **Управление состоянием**: Pinia (stores)
- **Стили**: Tailwind CSS + PostCSS + Autoprefixer
- **HTTP клиент**: Fetch API или Axios для запросов к бэкенду

### Backend (Node.js)
- **Серверная платформа**: Node.js
- **Основной файл**: `backend/index.js`
- **База данных**: MongoDB (см. MongoDB.md)
- **API**: RESTful API для фронтенда

## 🔄 Поток данных

```
Frontend Components
    ↓
Pinia Stores (authStore, cartStore)
    ↓
HTTP Requests (Fetch/Axios)
    ↓
Backend API (Node.js)
    ↓
MongoDB Database
```

## � Установка и запуск

### Предварительные требования
- Node.js 20.19.0+ или 22.12.0+
- npm или yarn
- MongoDB (локально или хостинг)

### Установка зависимостей Frontend
```bash
npm install
```

### Установка зависимостей Backend
```bash
cd backend
npm install
cd ..
```

### Запуск dev сервера Frontend
```bash
npm run dev
```
Приложение откроется на `http://localhost:5174`

### Запуск Backend сервера
```bash
cd backend
npm start
# или для разработки с автоперезагрузкой:
npm run dev
```
Сервер будет запущен на указанном порту (см. backend/index.js)

### Сборка для продакшена
```bash
npm run build
```

### Предпросмотр собранного приложения
```bash
npm run preview
```
